# Inventario de Fontes de Diagramas (Historico)

Documento historico de fontes que podem ser usadas como exemplos de teste visual.

## Objetivo
- Registrar fontes de diagramas para validacao manual do produto.
- Controlar licencas e restricoes de uso.

## Uso atual
- Referencia de exemplos para testes do fluxo (upload -> analise -> relatorio).
- Uso atual: referencia de testes visuais do fluxo.
